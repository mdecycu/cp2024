Examples

Brython 程式允許使用者採用 Python 語法在網頁上編寫程式, 當網頁載入時, 這些 Brython 程式碼透過 Brython 解譯程式庫轉為 Javascript 程式碼, 如此就可以直接在網頁上執行 (因為網頁瀏覽器中內建 Javascript 解譯器).

其實, 在網頁上編寫 Javascript 程式, 可以直接利用 HTML 語法中的 CANVAS 繪圖, 而 Brython 也可以採用相同的方式在網頁中進行靜態或動態繪圖.
